OpenFaaS (Function as a Service)

Combining a few of the tools mentioned previously (Docker, Docker Swarm, and Kubernetes), OpenFaaS is an open source project that allows you to easily build serverless functions. The framework helps automate the deployment of an API gateway to support your function and collected metrics. The gateway scales according to demand, and a UI is made available for monitoring purposes (https://github.com/openfaas/faas).
