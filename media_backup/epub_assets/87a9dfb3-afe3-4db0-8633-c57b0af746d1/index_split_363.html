Summary

In this chapter, we focused on the Microsoft Azure cloud platform in line with the CNMM model we described earlier. We started with the basics of Microsoft Azure, its brief history, and then deep into a few of its differentiating services and offerings. We then looked at the same sample serverless microservice application that we used in Chapter 14, Amazon Web Services, as well, and deployed it on Azure using Azure functions and proxies. After that, we focused on DevOps and CI/CD patterns and how those relate to both serverless and Docker container-based application models. Finally, we looked at Azure's cloud migration guidance, how it differentiates from AWS' messaging, followed by Azure Stack-related hybrid cloud aspects.
