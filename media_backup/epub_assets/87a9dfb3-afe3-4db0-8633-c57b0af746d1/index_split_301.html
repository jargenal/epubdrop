Kubernetes

With the popularity and success of Docker expanding in the mid 2010s, Google recognized the trend of containerization and contributed significantly to its adoption. Google designed Kubernetes from scratch to orchestrate Docker containers, basing it off of their internal system infrastructure orchestration system called Borg. Kubernetes is a powerful, open-source tool that helps mount storage systems, check application health, replicate application instances, autoscale, roll out updates, load balance, monitor resources, and debug applications, among many other capabilities. In 2016, Kubernetes was donated to the Cloud Native Compute Foundation, ensuring it would remain free to use and supported by an active community (https://kubernetes.io/).
