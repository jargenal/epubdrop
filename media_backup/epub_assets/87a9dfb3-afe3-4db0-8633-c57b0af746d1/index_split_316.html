AWS platform – differentiators

CNMM is a maturity model wherein if you are using only core building blocks from the cloud (such as, infrastructure components), then you are also cloud-native, but might be less mature than someone else who might be using services higher up the stack to actually get the full benefits of Cloud. So, based on this concept, if you have to truly leverage the power of AWS cloud, then you should focus on a few of the areas that are talked about in the following sections.
